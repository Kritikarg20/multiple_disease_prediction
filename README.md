# MediPredict AI – Multiple Disease Prediction System

---

##  Description

MediPredict AI is a machine learning-powered healthcare application that predicts the likelihood of multiple diseases based on user-provided medical parameters. The platform provides a simple and interactive interface where users can receive instant disease risk predictions using trained AI models.

---

## Problem Statement

Many diseases can be managed more effectively when identified at an early stage. However, access to preliminary health assessments is often limited. This project was developed to provide an accessible and user-friendly platform that leverages machine learning to assist users in evaluating potential disease risks based on health-related data.

---

##  Features

* Multi-Disease Prediction Platform
* Diabetes Prediction
* Heart Disease Prediction
* Kidney Disease Prediction
* Liver Disease Prediction
* Breast Cancer Prediction
* Parkinson's Disease Prediction
* Hepatitis Prediction
* Lung Cancer Prediction
* Interactive and User-Friendly Interface
* Real-Time Prediction Results
* AI-Based Health Assistance
* Secure Data Processing

---

## Dataset Used

The project utilizes publicly available healthcare datasets from trusted sources such as:

* Pima Indians Diabetes Dataset
* Cleveland Heart Disease Dataset
* Chronic Kidney Disease Dataset
* Indian Liver Patient Dataset (ILPD)
* Wisconsin Breast Cancer Dataset
* Parkinson's Disease Dataset
* Hepatitis Dataset
* Lung Cancer Dataset

These datasets contain patient health parameters used to train and evaluate machine learning models.

##  How to Run

### 1. Clone the Repository

```bash
git clone <repository-url>
cd Multiple_Disease_Prediction
```

### 2. Install Frontend Dependencies

```bash
npm install
```

### 3. Install Python Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Environment Variables

Create a `.env` file and add the required configuration values.

### 5. Start the Application

```bash
npm run dev
```

### 6. Access the Application

Open your browser and navigate to:

```text
http://localhost:5173
```
